# Archived Behavior Scripts

This folder contains older standalone MATLAB scripts for behavior analysis after DeepLabCut.

They are kept as historical references and backups for:

- Open Field Test (OFT)
- Novel Object Recognition / Novelty Preference Recognition (NPR)
- Zero Maze
- Y-maze
- Forced Swim Test (FST)
- Tail Suspension Test (TST)

For routine behavior analysis, use the standardized entry point in the repository root:

```matlab
SongLabDLC_behavior_analysis
```
